# Coral Skills

Agent skills for [Coral](https://withcoral.com) — one SQL interface over APIs, files, and live sources, built for agents.

## Installation

```bash
npx skills add withcoral/skills
```

## Available Skills

| Skill | Description |
|-------|-------------|
| `coral` | Query connected sources using Coral SQL |
